- **Полностью на русском языке** - весь контент на русском языке
- **Современный дизайн** - красивый UI с градиентами и анимациями
- **Адаптивная верстка** - работает на всех устройствах
- **Интерактивный поиск** - быстрый поиск по командам
- **Анимации** - плавные переходы и эффекты
- **Мобильная навигация** - удобное меню для мобильных устройств

## 🚀 Как использовать

### Локальный запуск
1. Скачайте все файлы в одну папку
2. Откройте `index.html` в любом современном браузере
3. Сайт готов к использованию!

### Размещение на сервере
1. Загрузите файлы на ваш веб-сервер
2. Убедитесь, что все файлы находятся в одной директории
3. Откройте сайт по адресу вашего сервера

## 📋 Команды Termux

Сайт содержит 20 основных команд:

1. **Обновление системы** - `pkg update && pkg upgrade`
2. **Установка пакетов** - `pkg install [пакет]`
3. **Удаление пакетов** - `pkg remove [пакет]`
4. **Поиск пакетов** - `pkg search [запрос]`
5. **Информация о пакете** - `pkg show [пакет]`
6. **Список установленных** - `pkg list-installed`
7. **Очистка кэша** - `pkg clean`
8. **Просмотр файлов** - `ls [опции]`
9. **Смена директории** - `cd [путь]`
10. **Создание папок** - `mkdir [папка]`
11. **Копирование файлов** - `cp [источник] [назначение]`
12. **Перемещение файлов** - `mv [источник] [назначение]`
13. **Удаление файлов** - `rm [файл]`
14. **Просмотр содержимого** - `cat [файл]`
15. **Редактирование файлов** - `nano [файл]` или `vim [файл]`
16. **Права доступа** - `chmod [права] [файл]`
17. **Владелец файла** - `chown [пользователь] [файл]`
18. **Поиск файлов** - `find [путь] [критерии]`
19. **Поиск текста** - `grep [шаблон] [файл]`
20. **Сетевые соединения** - `netstat [опции]`
21. **Процессы системы** - `ps [опции]`

## 🔧 Кастомизация

### Изменение цветов
Отредактируйте CSS переменные в `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #10b981;
    /* и другие цвета */
}
```

### Добавление новых команд
Отредактируйте массив `termuxCommands` в `script.js`:
```javascript
{
    title: "Название команды",
    description: "Описание",
    icon: "fas fa-icon",
    iconBg: "#цвет",
    syntax: "синтаксис",
    examples: ["пример1", "пример2"]
}
```
