# Мой веб-сайт на PHP, JavaScript и CSS

Современный веб-сайт, созданный с использованием PHP, JavaScript и CSS с подключением к базе данных MySQL.

## Особенности

- ✅ **PHP** - серверная часть с подключением к MySQL
- ✅ **JavaScript** - интерактивность и анимации
- ✅ **CSS** - современный адаптивный дизайн
- ✅ **MySQL** - хранение данных
- ✅ **AJAX** - асинхронная отправка форм
- ✅ **Адаптивный дизайн** - работает на всех устройствах

## Структура файлов

```
├── simple_demo.php      # Упрощенная демо версия (рекомендуется для тестирования)
├── index.php            # Полная версия с подключением к БД
├── config.php           # Конфигурация базы данных
├── demo_config.php      # Демонстрационная конфигурация
├── process_form.php     # Обработка форм
├── style.css            # Стили CSS
├── script.js            # JavaScript функциональность
├── test_db.php          # Тест подключения к БД
└── README.md            # Этот файл
```

## База данных

Сайт автоматически создаст таблицу `contact_messages` для хранения сообщений из контактной формы:

```sql
CREATE TABLE contact_messages (
    id INT AUTO_INCREMENT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    email VARCHAR(100) NOT NULL,
    message TEXT NOT NULL,
    created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP
);
```

## Кастомизация

### Изменение цветов
Отредактируйте CSS переменные в `style.css`:

```css
:root {
    --primary-color: #667eea;
    --secondary-color: #764ba2;
    --accent-color: #ff6b6b;
}
```

### Добавление новых секций
1. Добавьте HTML в `simple_demo.php` или `index.php`
2. Добавьте стили в `style.css`
3. При необходимости добавьте JavaScript функциональность

## Устранение неполадок

### Ошибка подключения к базе данных
- Используйте `simple_demo.php` для демонстрации
- Проверьте правильность данных в `config.php`
- Убедитесь, что MySQL сервер запущен
- Проверьте права доступа пользователя

### Форма не отправляется
- Проверьте консоль браузера на ошибки JavaScript
- Убедитесь, что файл `process_form.php` доступен
- Проверьте права на запись в базу данных

### Стили не загружаются
- Проверьте путь к файлу `style.css`
- Убедитесь, что веб-сервер настроен для обработки CSS файлов
