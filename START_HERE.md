# 🚀 Быстрый запуск сайта

## Что у вас есть

Полноценный веб-сайт на **PHP**, **JavaScript** и **CSS** с подключением к **MySQL** базе данных.

## ⚡ Быстрый старт (5 секунд)

```bash
# 1. Запустите PHP сервер
php -S localhost:8000

# 2. Откройте в браузере:
# http://localhost:8000/simple_demo.php
```

## 📁 Основные файлы

- **`simple_demo.php`** - Демо версия (работает без БД) ⭐ **НАЧНИТЕ С ЭТОГО**
- **`index.php`** - Полная версия с подключением к MySQL
- **`style.css`** - Красивые стили и анимации
- **`script.js`** - Интерактивность и AJAX

## 🎯 Что показывает сайт

✅ **Главная страница** - приветствие и кнопки  
✅ **О нас** - описание технологий  
✅ **База данных** - информация о MySQL  
✅ **Контакты** - форма с AJAX отправкой  
✅ **Адаптивный дизайн** - работает на всех устройствах  

## 🔧 Если нужна реальная база данных

1. Убедитесь, что MySQL сервер запущен
2. Создайте базу `alikf142_image`
3. Используйте `index.php` вместо `simple_demo.php`

## 🌐 Доступные URL

- **Демо:** http://localhost:8000/simple_demo.php
- **Полная версия:** http://localhost:8000/index.php
