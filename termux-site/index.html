Установка Termux

  1. Скачайте Termux из F‑Droid или из официального GitHub релиза.
  2. После первого запуска выполните: 
    pkg update && pkg upgrade
termux-change-repo
termux-setup-storage
  3. Установите базовые инструменты: 
    pkg install git curl wget nano vim tmux openssh

Примечание: не используйте sudo в Termux. Для рут-прав на рутованном устройстве доступен tsu.

Частые вопросы

Где находятся мои файлы?

Домашняя директория: /data/data/com.termux/files/home. Общая память доступна после termux-setup-storage в /storage/shared (или смонтирована как ~/storage).

Как изменить зеркала репозиториев?

Используйте termux-change-repo и выберите быстрые зеркала. После этого выполните pkg update.

Как запустить дистрибутив Linux?

Установите proot-distro, затем: proot-distro install ubuntu && proot-distro login ubuntu.
